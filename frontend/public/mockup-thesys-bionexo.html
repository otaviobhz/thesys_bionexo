API do Thesys (para o time Thanner/Gabriel)

O Thesys precisa expor 4 endpoints REST para o portal funcionar:

GET /api/bionexo/itens Produtos Ativos

Lista produtos ativos para pareamento de SKU.

Query Params

ParamTipoObrigatórioDescrição
searchstringNãoFiltrar por SKU ou descrição

Response

[
  { "id": 1, "sku": "9928", "descricao": "SERINGA 10ML LUER LOCK", "unidade": "CX" },
  { "id": 2, "sku": "5512", "descricao": "GAZE ESTERIL 13F PCT C/10", "unidade": "PCT" }
]

SQL

SELECT Id_Item AS id, Codigo AS sku, descricao, u.codigo AS unidade
FROM Itens i LEFT JOIN Unidades u ON u.id_unidade = i.id_unidade
WHERE i.ativo = 'S' AND i.excl_data IS NULL
AND (Codigo LIKE '%search%' OR descricao LIKE '%search%')
GET /api/bionexo/precos/:idTabela Tabela de Preços

Retorna preços de uma tabela específica.

Response

[
  { "sku": "9928", "descricao": "SERINGA 10ML", "preco": 12.50 },
  { "sku": "5512", "descricao": "GAZE ESTERIL", "preco": 8.90 }
]
GET /api/bionexo/hospitais Hospitais

Lista hospitais cadastrados (CliFor com hospital='S').

Response

[
  { "id": 1, "razao_social": "Hospital Albert Einstein", "nome_fantasia": "Einstein", "cnpj": "60.765.823/0001-30" }
]
POST /api/bionexo/cotacao Criar Cotação

Cria cotação no Thesys após envio ao Bionexo.

Request Body

{
  "cod_clifor": 123,
  "id_pagamento_condicoes": 1,
  "prazo_entrega": 5,
  "fat_minimo": 500.00,
  "itens": [
    { "id_item": 9928, "quantidade": 500, "preco_unitario": 12.50, "observacao": "Entrega em 5 dias" }
  ]
}

Response

{ "id_venda_cotacao": 456, "numero": 10293 }

API da Bionexo (EDI WebService SOAP)

OperaçãoCódigoDireçãoDescrição
Download CotaçõesWGG← BionexoReceber novas cotações
Responder CotaçãoWHS→ BionexoEnviar preços/produtos
Alterar/CancelarWHU→ BionexoModificar ou cancelar resposta
ProrrogaçõesWGA← BionexoPrazos alterados
Pedidos ConfirmadosWJG← BionexoDownload de pedidos gerados

Checklist de Configuração

  • Subir Docker Compose (docker compose up -d)
  • Login com admin@promeho.com.br / Admin123!
  • Receber URL + Token da API Thesys (Thanner/Gabriel)
  • Configurar API Thesys na tela /config
  • Testar conexão Thesys
  • Solicitar token Bionexo WebService
  • Configurar Bionexo na tela /config
  • Testar conexão Bionexo (sandbox)
  • Cadastrar palavras-chave iniciais
  • Fazer o primeiro pareamento (De-Para)
  • Ativar bot automático
  • Testar fluxo completo no sandbox
  • Trocar para ambiente Produção
